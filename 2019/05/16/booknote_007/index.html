读书笔记——学会提问

发表于 2019-05-16 | 分类于 读书笔记 | 本文阅读总量 次
字数统计: 1.2k 字 | 阅读时长 ≈ 4 分钟
  • 学会提问
    • 批判性思维
      • 维度
        • 意识到他们是一整套环环相扣的关键问题
        • 有能力在适当的时机以适当的方式提出和回答这些问题
        • 积极主动地使用这些关键问题的强烈渴望
      • 分类
        • 弱势批判性思维
          • 捍卫自己的信念
        • 强势批判性思维
          • 对自己当初的信念加以评估和修正
      • 主要价值观
        -自主决断
        • 好奇心
        • 谦恭有礼
        • 发自内心的尊重严密的论证
      • 障碍
        • 晕轮效应
        • 信念固着
          • 确认性偏见
        • 可得性启发
          • 近因效应
        • 答非所问
        • 一厢情愿
          • 奇迹式思维
    • 思维方式
      • 海绵式思维
        • 被动
        • 强调知识获取的结果
      • 淘金式思维
        • 主动
        • 重视知识获取的过程
        • 互动性参与
        • 精神检查表
          • 我有没有问“为什么”别人要我相信某件事
          • 在我想别人所说的话可能存在的问题时，我没有把它们记录下来
          • 我有没有评估别人说过的话
          • 针对某一主题，我有没有基于别人所说的合理内容来形成自己的结论
    • 批判性评价
      • 找出结论
      • 找到支撑结论的理由
      • 找关键词线索
        • 检查论题，看有没有关键词
        • 在理由和结论中寻找关键词或短语
        • 留心抽象的词或短语
        • 通过反串来判断别人怎样给特定的词或短语下不同的定义
      • 寻找假设
    • 论题
      • 引起对话或讨论的问题、争议
      • 分类
        • 描述性论题
          • 指针对过去、现在或将来的各种描述的精确与否提出的问题
          • 是不是/是什么
        • 规定性论题
          • 指针对什么该做什么不该做，什么是对什么是错，以及什么是好什么是坏所提出的问题
          • 应不应该
    • 证据
      • 立论者告知的明确信息，用来支撑或证明一个事实断言的可靠性
      • 类型
        • 直觉
        • 个人经历
        • 典型案例
        • 当事人证言
        • 专家意见
        • 个人观察
        • 研究报告
        • 类比
    • 谬误
      • 论证中的欺骗手段
      • 类型
        • 人身攻击型谬误
          • 针对个人的人身攻击，而不是反驳其提供的理由
        • 滑坡谬误
          • 设想采取提议的步骤会引发一连串不可控的不利事件，而实际上却有现成的程序可用来防止此类连锁事件的发生
        • 追求完美解决方案谬误
          • 错误的认为因为尝试某种解决方案后还有遗留问题为解决，那么这种解决方案根本就不该采用
        • 诉诸公众谬误
          • 通过引述大部分人都持有这一观点的说法，试图证明某个断言有道理；错误的以为大部分人喜欢的事就是可取的
        • 诉诸可疑权威谬误
          • 引用某一项权威的话来证明结论，而该权威对这一论题并没有特别的专门知识
        • 诉诸感情谬误
          • 使用带强烈感情色彩的语言来分散读者或听众的注意力，让他们忽视相关的理由和证据
        • 稻草人谬误
          • 歪曲对方观点，使它容易受到攻击
        • 虚假的两难选择谬误
        • 乱扣帽子谬误
          • 错误的以为因为你给某个特定事件或行为起了名字，你就合理解释了这一件事
        • 计划谬误
        • 光环效应谬误
        • 转移话题谬误
        • 循环论证谬误
        • 以偏概全谬误
        • 强求确定性谬误
          • 认为一个研究结论不是百分百确定的话，就应该被抛弃
        • 错误类比谬误
        • 过度简化因果关系谬误
        • 因果混淆谬误
        • 忽略常见原因谬误
        • 事后归因谬误
          • 假设某件事A是由另一件事B所引发，仅仅因为事件A在时间上紧随在事件B之后发生
        • 基本归因谬误
    • 假设
      • 一个信念，通常没有明说出来，而作者认为是理所当然，并用来证明外在的论证
      • 分类
        • 价值观假设
          • 在特定情形下没有明说出来的喜欢一种价值观超过另一种价值观的倾向
        • 描述性假设
          • 一种关于世界过去、现在和将来是怎么样的没有明说的信念
    • 论证
      • 论证必有其目的
      • 基本构成要素
        • 结论
          • 他人试图传递给你的信息
        • 理由
          • 指我们为什么要相信某个结论的解释或依据
          • 类型
