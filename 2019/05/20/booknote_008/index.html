    • 分析
      • SCQA
        • 主角-Protagonist
          • 具体描述当事者的价值观、具体特色的行动准则等
        • 状况-Situation
          • 描述目前安定稳定的状态，无论好坏
        • 问题-Complication
          • 假设一个事件或障碍，颠覆稳定的状态
        • 课题-Question
          • 针对这个问题，假设一个对主角而言最重要的疑问
        • 回答-Answer
          • 提出解决的手段，必须具有说服力
      • 可行性研究
        • 只有单一提案时
      • 情景分析
        • 制作脚本

            • 五力

              • 因子统合

      • 3C/4C/5C——思考事业战略
        • Company-自家公司分析
          • 市场占有率
          • 技术力
          • 销售力
          • 成本竞争力
          • 品牌力
          • …
        • Competitor-竞争对手分析
          • 市场占有率
          • 强项
          • 弱点
          • 战略
          • …
        • Customer-顾客和市场分析
          • 规模/成长率
          • 市场屏障
          • 需求/属性
          • 结构/变化
          • 未来展望
          • 消费决策过程
          • 消费决策者
          • …
        • Channel-渠道
        • Controller-当局
      • 五力——行业分析
        • 目前的产业竞争
          • 业界的特征
          • 竞争态势的变化
        • 新加入者的威胁
          • 规模经济、品牌力、投资额、差异化等进入门槛
        • 替代品的威胁
          • 会不会出现替代品夺走目前需求
        • 供货商的议价能力
          • 变化
        • 购买者的议价能力
          • 转变
      • 7S——思考组织策略
        • Strategy-经营策略
          • 自家公司固有的竞争模式
        • Structure-组织结构
          • 公司内部的团体
        • System-运营系统
          • 管理会计、人事制度等
        • Style-经营风格
          • 决策的过程等
        • Staff-职员
          • 个人的能力和资质
        • Skill-组织技能
          • 组织全体的力量
        • Shared Value-共享价值观
          • 企业理念、使命感
      • 职业生涯矩阵
      • 企业价值创造矩阵——检讨企业并购
      • 产品·市场矩阵/成长矩阵——思考成长战略
        • 市场渗透
        • 市场开发
        • 产品开发
        • 多元化
      • 4P——拟定营销策略
        • Product-产品
          -顾客价值的观点
        • Price-价格
          • 与竞品或服务的平衡点
          • 行情
          • 消费者的观点
        • Promotion-促销
          • 传递商品价值给用户
        • Place-渠道
          • 物流战略
      • 促销组合
        • 宣传广告
          • 电视
          • 电台
          • 报纸、杂志
          • 网络广告
        • 公关广告
        • 营业推广
        • 人员销售
      • AIDMA模型——分析消费决策流程
        • Attention-注意
          • 吸引消费者注意，传递产品和服务的讯息
        • Interest-兴趣、关心
          • 让消费者对产品或服务产生兴趣、关心
        • Desire-欲望
          • 让消费者燃起想要使用产品和服务的欲望
        • Memory/Motivate-记忆/动机
          • 让消费者记住产品
        • Action-行动
          • 实际行动，购买产品和服务
      • 商业系统（价值链）
